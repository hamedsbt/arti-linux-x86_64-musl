Heads up

Important: This will connect using , which is for demonstration purposes only. It has limited capacity and may be removed at any time. You'll need to host your own gateway if you want to deploy this solution for your users.

The gateway is not doing any of the tor protocol - it's only required for TCP. In NodeJS you can run tor-js without a gateway because it can open TCP sockets directly.
